HTML Page Structure:

HTML page consists of two main parts:

First :Head section starts with <head >opening tag and ends with </head>closing tag.
head section contains information specific to the page including title tag,
meta tag and links to both style sheets and Script 
Second: body section starts with <body >opening tag and ends with </body>closing tag.
body section contains the informations the client will see including headers,images,buttons 
and paragraphs

Program Example

Html document must be saved with html extension.

Html page starts with <!DOCTYPE html > declaration statement, to declare that the document is an HTML5 document

<html lang="en"> is the start of the document , lang attribute states the document language which is English

< meta charset = "utf-8" /> is a meta tag to declare that the document
is in utf-8 code (more information about meta tags and utf-8 code will be in next sections)

< title > specifies title for the html page this title will be shown in the browser's title bar

< h1 > -< h6 > tags display headings of different sizes h1 the largest h6 the smallest

< p > element defines a paragraph
